# Netflix-Content-Performance-Audience-Insights-Analysis
Netflix Movie Analytics: Understanding Genre Trends and Audience Preferences

The objective of this project is to:
Analyze movie content trends
Understand audience engagement patterns
Explore relationships between popularity and ratings
Identify genre-level performance
Provide business recommendations for content investment

Tools Used:
Python
Pandas (data cleaning & manipulation)
Matplotlib
Seaborn


Business Problem:
Streaming platforms rely on audience engagement and content quality to retain viewers. This project analyzes movie-level data to identify genre performance, release trends, and audience behavior patterns that can support content strategy decisions
